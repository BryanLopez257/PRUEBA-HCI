# Wireframes Mid-Fidelity

Descripción y anotaciones para prototipos de mediana fidelidad.

1) Login (Mid-Fi)
- Header con logo y título
- Formulario centrado con hint y validación inline

2) Dashboard (Mid-Fi)
- Sidebar colapsable con íconos + etiquetas
- Header con breadcrumbs y acciones rápidas
- Grid de cards con métricas y gráficas pequeñas

3) Formulario multi-paso
- Stepper horizontal con estado activo
- Validaciones en cada paso y mensajes de error

Anotaciones de interacción:
- Estado activo de menú
- Loader al cargar reportes
- Tooltips para términos técnicos
